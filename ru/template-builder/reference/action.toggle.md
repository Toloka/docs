# action.toggle

Компонент меняет значение в данных с `true` на `false` и наоборот.

## Свойства компонента {#properties}

| Название                                 | Тип                                                                                  | Описание                                                                            |
| ---------------------------------------- | ------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------- |
| `type`<span style="color: red">\*</span> | "action.toggle"                                                                      | <p>Задает тип компонента.</p>                                                       |
| `data`                                   | <a class="xref popup-link" href="../concepts/types.dita#types/writable">writable</a> | <p>Данные, в которых значение будет изменено. Тип данных должен быть `boolean`.</p> |
