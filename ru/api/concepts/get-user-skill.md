## Запрос {#request}

{% list tabs %}

- Боевая версия

    ```bash
    GET https://toloka.dev/api/v1/user-skills/<id>
    Authorization: OAuth PlaceYourRealOAuthToken_Here
    ```

- Песочница

    ```bash
    GET ihttps://sandbox.toloka.dev/api/v1/user-skills/<id>
    Authorization: OAuth PlaceYourRealOAuthToken_Here
    ```

{% endlist %}
